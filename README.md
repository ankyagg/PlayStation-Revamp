# Playstation Web Experience Revamp

![Playstation UI Revamp](https://playstation-revamp.vercel.app/og-image.jpg)

An exquisite, fan-made recreation and reimagination of the core PlayStation Console User Interface designed entirely for the web. Explore the immersive dashboard, test controller connectivity, and navigate horizontally through slick custom animations natively!

### 🌐 **[Live Deployment Link](https://playstation-revamp.vercel.app/)**

## ✨ Features
- **DualSense Mobile Connect**: Connect your smartphone by scanning the QR code and use it as a virtual controller via WebSockets payload.
- **Immersive 3D Controller Showcase**: A dynamic, rendered 3D DualSense component interacting natively with input using `@react-three/fiber`.
- **Hardware Store Carousel**: Use pointer arrows (or keyboard/controller) to navigate a slick, interactive accessories section.
- **Fluid Dashboard Navigation**: Seamless UI interaction styled akin to actual console dashboard libraries.
- **High Performance Design**: Responsive implementation built deeply with Tailwind CSS and GSAP for lush, butter-smooth micro-animations.

## 🛠️ Tech Stack
- React + Vite
- GSAP Animation Engine
- Tailwind CSS (+ modules)
- WebSockets (`useDualSenseWS` custom endpoint control)

## ⚙️ Running Locally
1. Clone repository
2. **Install Dependencies**: `npm install`
3. **Start Development Server**: `npm run dev`
4. Deploy locally and launch terminal URL.
